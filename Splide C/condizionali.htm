Controlli condizionali:

 

Le istruzioni condizionali in C sono:

If-else
Operatore ternario ?
switch

if - else

l'istruzione if-else ha la seguente sintassi:

if (espressione)
   istruzione1
else
   istruzione2

Facciamo un esempio, assumendo che valore della variabile intera "risultato_esame" ha un valore compreso tra 0 e 30:

 

if (risultato_esame >=18)
   printf ("Complimenti hai superato l'esame");
else 
    if (risultato_esame >=15) 
       printf ("Devi sostenere l'orale");
else
      printf ("Non hai superato l'esame");

Dopo un if o un else viene eseguita solo la prima istruzione, se si vuole eseguire un blocco di  istruzioni esse devono essere racchiuse tra una graffa di apertura ( { )ed una graffa di chiusura ( } )

 

int passato;
if (risultato_esame >=18)
 { printf ("Complimenti hai superato l'esame");
   passato = 1;
 }
else
{ printf ("Non hai superato l'esame");
   passato = 0;
}

Operatore Ternario

L'operatore ternario "? :" è una forma sintetica dell'istruzione if-else,.
La sintassi dell'operatore è la seguente:
espressione1 ? espressione2 : espressione3

// Corrispondente blocco if-else
if (espressione1)
 {espressione2
 }
else
 {espressione3
 }

 

Quindi se l'espressione1 risulta vera, si esegue l'espressione2, altrimenti si esegue l'espressione3.

Per esempio il seguente codice assegna alla variabile "max" il massimo tra "alfa" e "beta":

 

if (alfa > beta)
   max = alfa;
else
   max = beta;

// che corrisponde a...
max = (alfa>beta) ? alfa : beta;

switch

switch può essere usata solo mìnei casi dove:

Viene valutata solamente una variabile, tutte le scelte dipenderanno, infatti, da questa variabile. La variabile deve essere un tipo int, short, long o char;

Ogni singolo valore della variabile può controllare solo una scelta. Una scelta finale, chiamata default è incaricata di catturare tutti i valori dei casi non specificati;
L'istruzione switch ha la seguente struttura:
switch (espressione) {
case elem1:istruzione1;break;
case elem2:istruzione2;break;
...
case elemn:istruzionen;break;
default:istruzione;break;
}

 

Lo switch ha una struttura  a cascata, cioè se eseguo il primo caso (avendo sotto degli altri) e non metto un break per uscire, continua ad eseguire anche le istruzioni successive che fanno parte di altre casistiche

Ad esempio  assumendo di avere la variabile intera "numero" che può assumere valori maggiori od uguali a zero:

switch (numero) {
case 0:printf("Zero"); break;
case 1:printf("Uno");break;
case 2:printf("Due");break;
case 3:
case 4:
case 5: printf("Valore positivo piccolo");break;
default: printf("Valore positivo grande");break;
}
