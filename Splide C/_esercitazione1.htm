Esercizio 1

Preparare un programma che conti il numero di caratteri digitati fino ad al carattere '$' che terminerà il programma, usando getchar.

#include <stdio.h>
main()
{
int ch, i = 0;
while((ch = getchar()) != '$')
i ++;
printf("%d\n", i);
}

Esercizio 2

Preparare un programma che trasformi i caratteri in minuscolo in input in caratteri maiuscolo usando getchar e putchar.

#include <ctype.h> /* Per la definizione di toupper */
#include <stdio.h> 
void main()
{
int ch;
while((ch = getchar()) != '$')
putchar(toupper(ch));
}
