Esercizio 1(25/30)

Si simuli un Fast Food con tre casse e tre cassieri/camerieri e tre possibili tipologie di cibo panini, patatine, bibbite. Mentre i panini sono contenuti in tre distributori uno per cassa, le porzioni di patatine sono contenute in due distributori e le bibite in un unico frigo. Questo comporta che i cassieri che devono soddisfare una ordinazione con patatine o bibbite devono assicurarsi la disponibilità di uno dei distributori di patatine e del frigo.

La vita di ogni cassiere è :

1.   Finchè non è ora di chiusura

2.   Arriva un cliente

3.   prende ordinazione che consiste in tre interi dati in maniera random rispettivamente il numero di panini, di porzioni di patatine, e di bibbite

4.    Il cassiere registra l’ordinazione e la soddisfa.

5.   Torna al punto 2

Al termine si devono stampare per ogni cassa il numero di panini di bibite e  porzioni di patatine vendute

Sviluppare in linguaggio C un programma che simuli con i thread la situazione descritta, commentando le scelte fatte.

Esercizio 2(5/30)

Scrivere un programma C sotto SO linux che riceva in input da

tastiera due numeri interi, a e b, e ne stampi a video:

– la somma "a+b" solo quando riceve il segnale SIGUSR2

– la differenza "a-b" quando riceve il segnale SIGUSR1.

 Il programma esce quando riceve SIGINT.

 

 


 
Esercizio 2
#include <stdio.h>
#include <stdlib.h>
#include <signal.h>
 
int a=0, b=0, c=0;
int esci=1;
 
void handler (int segnale) {
        if (segnale == SIGUSR2) {
               c = a + b;
               printf ("La somma di %d + %d e' %d\n", a, b, c);
        } else {
               if (segnale == SIGUSR1) {
                       c = a - b;
                       printf ("La differenza di %d - %d e' %d\n", a, b, c);
               }
                else {
                       if (segnale == SIGINT) {
                               esci = 0;
                       }
               }
        }
}
 
int main (int argn, char *args[]) {
        printf ("Inserire il primo intero: ");
        scanf("%d", &a);
        printf ("Inserire il secondo intero: ");
        scanf("\n%d", &b);
 
        if (signal (SIGUSR2, handler) == SIG_ERR) {
               perror ("Impossibile intercettare SIGUSR2\n");
               exit (-1);
        }
 
        if (signal (SIGUSR1, handler) == SIG_ERR) {
               perror ("Impossibile intercettare SIGUSR2\n");
               exit (-1);
        }
 
        if (signal (SIGINT, handler) == SIG_ERR) {
               perror ("Impossibile intercettare SIGUSR2\n");
               exit (-1);
        }
 
        while ( esci ) {
               pause();
        }
 
        return 0;
}
