Gli operatori  & ed | agiscono operando l'AND e L'OR bit per bit 

Ad esempio 

int a=3;  //0..0011

int b=5;  //0..0101

int c=a&b;  // c=1;

c=a|b   // c=7

a<<2   //0..1100 

b>>2  //0..0001

 

L'operatore & ha una doppia valenza, se usato come operatore prima di una variabile calcola l'indirizzo della stessa.

 

Esistono poi degli operatori ai quali bisogna porre particolare attenzione,

 

l'operatore di incremento (++)

l'operatore di decremento (--)

 

essi possono essere preposti o posposti alla variabile; se sono preposti il valore è calcolato prima che l'espressione sia valutata, altrimenti viene calcolato dopo;

Add esempio:

int x = 2; printf("%d \n", x++);  //  2
int y = 5;printf("%d \n", ++y); // 6
int z = 9;printf("%d \n", z--); // 9
int k = 6;printf("%d \n", --k); // 5
quindi

x++   equivale x = x + 1;

x+=1 equivale x = x + 1;

Esempi

int y = 4;
y += 2;  // y adesso vale 6
int x = 3;
x *= y + 3;

// x adesso vale 27 questo perché equivale a x=x*(y+3) e non x=(x*y)+3

 

Le conversioni

In C le conversioni da intero a float o double e viceversa sono automatiche quindi si può assegnare ad una varibile intera il valore di una varibile reale, essa la convertirà in intero automaticamente.

Ad esempio

int a;

float c;

a=10;c=a; // il valore di c sarà 10.0000

c=10.3;a=c; // il valore di a sarà 10

Lo stesso non succede con la divisione;

int a,b;

float c;

a=2; b=5;

f=b/a; // il valore di f sarà 1.00 perchè la divisione è tra due interi e quindi il risultato è un intero

Se si vuole il risultato in un altro formato di deve castomizzare;

f=(float)b/a; // il valore di f sarà 0.40000
