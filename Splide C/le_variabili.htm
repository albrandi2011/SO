Una variabile è un’astrazione di un’area di memoria

Una variabile è caratterizzata da

  • Nome
  • Tipo: specifica la classe di valori che la variabile può assumere (e quindi gli operatori applicabili al valore in essa contenuto)
  • Valore (di un determinato tipo)

In C esistono vari tipi di variabili, questo per venire incontro sia all'esigenze di rappresentabilità di grossi numeri sia al maggior risparmio di memoria possibile, utilizzando di volta in volta il tipo più adeguato ad una specifica situazione. Per usare una variabile si deve

• Dichiarare: si introduce una nuova variabile nel programma. Si dichiara il tipo e si alloca la memoria. Esempi

– int i, j; /* dati di tipo intero */

– float f; /* dati di tipo reale */

• Inizializzare:

– int i = 0;

• Usarla nel corso del programma. Ad esempio,

i=2;j=3;
