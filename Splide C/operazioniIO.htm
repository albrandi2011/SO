Strumenti adatti ad operare sull'input/output

 

Per adoperare le funzioni base di I/O si deve includere il file <stdio.h> che mette a disposizione alcune funzioni predefinite per eseguire la lettura da un dispositivo di input (es. tastiera) o scrittura su un dispositivo di output (es. video);

Le funzioni di cui parleremo sono

getchar
putchar
printf
scanf

getchar

getchar  legge un carattere per volta dalla tastiera; Ad esempio il programma mostrato conta il numero di volte che si è digitato un carattere

 (per interrompere il programma premere $).

#include <stdio.h>

main()
{
int ch, i = 0;

while((ch = getchar()) != '$')
i ++;

printf("%d\n", i);
}

putchar

putchar legge un carattere alla volta e lo stampa a video; 

Ad esempio il programma seguente converte ogni carattere inserito nell'equivalente lettera maiuscola grazie alla funzione toupper().

#include <ctype.h> /* Per la definizione di toupper */
#include <stdio.h> 

main()
{
int ch;

while((ch = getchar()) != '$')
putchar(toupper(ch));
}

printf


printf è l'istruzione per stampare a video più usata . La struttura di printf è la seguente:


int printf(char *formato, lista argomenti ...)

che stampa sullo stdout (il video in questo caso) la lista di argomenti conformemente alla stringa di formato specificata.

La funzione ritorna il numero di caratteri stampanti. 

La stringa di formato ha due tipi di argomenti:

1. caratteri ordinari:

questi vengono copiati nell'output;

2. specificazioni di conversione

contraddistinte dal carattere "%" e da un altro carattere che specifica il formato con il quale stampare le variabili presenti nella lista di argomenti

 

La tabella seguente mostra i possibili formati 

Stringa di controllo

  Cosa viene stampato
%d, %i Intero decimale
%f  Valore in virgola mobile(float o double)
%c Un carattere
%s  Una stringa di caratteri
%o Numero ottale
%x, %X Numero esadecimale
%u Intero senza segno
%e, %E Formato scientifico
%% Stampa il carattere %


È possibile inserire, tra il simbolo % e l'identificativo del formato, una delle seguenti voci:

Segno meno (-) esegue la giustificazione a sinistra
Un numero intero specifica l'ampiezza del campo
una coppia di interi divisi da un punto m.d dove m è l'ampiezza del campo e d è la precisione del numero; usato generalmente per le stringhe o per un numero di tipo reale.

Facciamo alcuni  esempi

int x = 40;
printf("Il numero è %d", x);  // "Il numero è 40"

printf("Il carattere ASSCI %d  è %c", x,x);  

// "Il carattere ASSCI 40  è ( 

print("%2.3f \n", 24.392734); // 24.393

printf("IVA = 20%%");//stampa "IVA = 20%"

 

All'intero  del formato si possono inserire delle sequenza di escape che servono per rappresentare dei caratteri "speciali". Le sequenze di escape iniziano con il carattere backslash (\) e sono interpretate come un singolo carattere.

Opzione  Descrizione
\n
 Ritorno a capo
\t
Tabulazione orizzontale
\b
 Tabulazione verticale
\a
Torna indietro di uno spazio
\f
 Salto pagina

scanf

scanf serve per leggere dallo stdin (generalmente la tastiera) una sequenza di caratteri (lettere o cifre) che verranno memorizzate all'interno di opportune variabili. 

int scanf(char *formato, lista argomenti ...)

A differenza della printf, la variabile deve essere messa preceduta dal simbolo &, perché in realtà tra gli argomenti non dobbiamo passare il nome della variabile, ma il suo indirizzo

#include <stdio.h>

int main()
{
int i;
scanf("%d ", &i);
printf("%d \n", i);
}

 

Questo semplice programma  legge un numero da tastiera e lo ristamparlo a video;
